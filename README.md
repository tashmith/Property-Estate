# Estate Agent App

# Project Overview
The Estate Agent App is a React-based web application for browsing property listings. It demonstrates React component structure, custom hooks, and proper project organization. The project focuses on clean code, reusable UI components, and simple data management using JSON.

---

# Main Features
- **Browse Properties:** View a list of properties with details like price, bedrooms, location and type.
- **Featured Properties:** Highlight important listings on the homepage.
- **Reusable UI Components:** Includes HeroSection, FavouritePanel, PropertyCard, and SearchBar etc......
- **Responsive Design:** Works well on mobile and desktop devices.
- **Custom Hooks:** useMobile for responsive behavior, useToast for notifications.
- **Data Management:** Property data is stored in properties.json for easy updates.
- **Testing:** Basic component tests implemented with JEST.
- **Security:** JSX encoding and basic CSP measures implemented.
